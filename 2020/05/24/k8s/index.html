2.基本环境配置

1.关闭selinux
setenforce 0临时关闭
sed -i “s/SELINUX=enforcing/SELINUX=disabled/g” /etc/selinux/config永久关闭
2.关闭swap分区或禁用swpa文件

注释掉关于swap分区的行

swapoff -a关闭交换分区

修改网卡配置

k8s环境中不可避免的需要进行网络转发或者流量转发的操作
vim /etc/sysctl.conf

使用sysctl -p完成应用修改

出现没有文件或目录不用管它，因为还没有生成，之后的配置过程中会生成这个文件

3.启用内核模块

vim /etc/sysconfig/modules/ipvs.modules 文件默认不存在，属于新建一个文件
写入下面5行内容
modprobe – ip_vs
modprobe – ip_vs_rr
modprobe – ip_vs_wrr
modprobe – ip_vs_sh
modprobe – nf_conntrack_ipv4
保存退出
执行命令来临时启用

查看命令是否启用

安装kubelet kubeadm kubectl（安装时版本1.18.03）

启动kubelet服务

此时用systemctl status kubelet查看服务状态，服务状态因为Error(状态码255)
使用journalctl -xe查看错误信息
可以看到failed to load Kubelet config file /var/lib/kubelet/config.yaml
原因是还没搭建k8s集群，所以config.yaml配置文件还不存在，后续创建后重启就能修复
